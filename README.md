# mysuni.chatGPT.project

써니C 챗봇 구현 서비스
- 현직자와의 인터뷰를 통해 현업에서의 문제점을 파악하고 이를 해결하기 위한 챗봇 서비스 아이디어 도출 과제
<br>

## 프로세스
<br>
playgound로 api 받기 -> 닷홈 웹호스팅 -> 파일질라 연동 -> 소스코드의 프롬프트 수정 -> 원하는 챗봇 서비스 웹 구현
<br>
<br>
<br>

## 결과물
**1) 블루버디**

- 리스트 정리에 도움
- 엔지니어용 서비스
<br>
(프롬프트에 여러 문장이 쌓일 수록 답변의 이상이 생기기에 간단한 명령과 하나의 예시만 추가)
